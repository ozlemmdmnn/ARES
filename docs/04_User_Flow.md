# User Flow
## Overview

This document describes how users interact with ARES.

Each flow represents a real-world scenario and defines how the application should behave.

The purpose is to provide a consistent user experience throughout the system.

## User Types
### Citizen

- Report disasters
- View safe zones
- Receive AI recommendations
- View emergency resources

---

### Emergency Team

- Manage incidents
- Update rescue status
- Coordinate teams
- Monitor resources

---

### Administrator

- Manage users
- View reports
- Monitor the entire system
- Configure settings
